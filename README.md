# 2026_1_CALCULO-APLICADO_1ER-PROYECTO

Primer proyecto del curso de Cálculo Aplicado (2026_1) sobre Comparación de Métodos de Integración Numérica.

El código se divide en partes:

- [x] Parte 1 Convergencia de Sumas Inferior y Superior

- [x] Parte 2 Influencia del Tipo de Partición

- [x] Parte 3 Comparación de Métodos de Integración Numérica

- [x] Parte 4 Integración de Montecarlo

En cada carpeta de cada parte está:

- el código Python a ejecutar,

- el CSV con los datos de la tabla, obtenidos al ejecutar el código

- y un archivo Excel con la gráfica a partir del CSV.

Nota. En la parte 3, el código ejecuta adicionalmente una función que grafica los rectángulos de aproximación para los tres tipos de partición.
